# Micro-Controllers-course-HIT
A comprehensive collection of lab exercises and projects from the Microcontrollers Laboratory course (50193) at Holon Institute of Technology (HIT). Features C-based development for embedded systems, covering interrupts, timers, ADC/DAC, and serial communication.

# Microcontrollers Laboratory (50193) - HIT

[cite_start]This repository contains the laboratory exercises and projects for the **Microcontrollers Laboratory** course at **Holon Institute of Technology (HIT)**[cite: 8, 9]. [cite_start]The course focuses on the architecture and operation of microcontrollers, emphasizing the bridge between hardware and software through practical development[cite: 14].

## 📖 Course Description
[cite_start]The course covers the principles of microcontroller architecture, including memory, CPU, ALU, and instruction encoding[cite: 12]. [cite_start]It provides hands-on experience in interfacing processors with memory and I/O peripherals, using polling and interrupts, and implementing serial communication protocols[cite: 12].

## 🛠️ Topics Covered
[cite_start]The repository is structured according to the course syllabus topics[cite: 16, 17]:
* [cite_start]**Introduction & Architecture:** Basics of MCU operation and development kit environment[cite: 16].
* [cite_start]**C Programming for MCUs:** Instruction encoding and software development in C[cite: 16].
* [cite_start]**I/O Operations:** Interfacing with physical input/output components[cite: 16].
* [cite_start]**Interrupts:** Implementation of hardware/software interrupts vs. polling[cite: 16].
* [cite_start]**Timers & Counters:** Working with internal timing modules[cite: 17].
* [cite_start]**Data Acquisition (ADC/DAC):** Analog-to-digital and digital-to-analog conversion processes[cite: 17].
* [cite_start]**Serial Communication:** Protocols including UART, I2C, and SPI[cite: 17].
* [cite_start]**Displays:** Interfacing with LCD modules[cite: 17].

## 💻 Technical Environment
